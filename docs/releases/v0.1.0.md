# v0.1.0 · First Public Open Source Release

发布日期：2026-03-08

## Release Summary

`v0.1.0` 是项目首个正式公开版本。这个版本的目标不是简单把代码推送到 GitHub，而是把仓库整理成一个可以阅读、可以部署、可以二次开发、可以持续维护的开源项目。

这一版主要完成了三类工作：

1. 产品能力收口
2. 自托管部署链路固化
3. 公开仓库文档、隐私与社区基础设施整理

## Highlights

### 1. 导师联系与准备流程工作台

项目已经覆盖从导师信息维护、联系状态跟踪、面试安排、准备笔记到材料沉淀的一整套核心链路，适合保研 / 考研 / 复试准备场景持续使用。

### 2. 文书与资料集中管理

支持在线维护简历、联系记录、准备笔记和资料链接，减少文件分散在本地文件夹、聊天记录和浏览器收藏夹中的问题。

### 3. 学校 / 研究所官方源库

支持学校与研究所检索，并提供官方源库快照恢复链路，方便在低内存服务器或离线场景中快速恢复基础数据。

### 4. 单机可落地的部署方案

项目提供 Docker Compose、自定义热发布脚本与 Caddy HTTPS 代理方案，适合个人或小团队在低成本服务器上持续迭代。

### 5. 开源仓库整理完成

公开版本已移除服务器连接信息、私有域名、敏感运维记忆和本地凭据痕迹，并补齐 README、CHANGELOG、MIT License、Issue 模板、PR 模板和社区规范文件。

## Included Capabilities

- 邮箱注册、登录、验证码找回密码
- GitHub OAuth 登录 / 注册
- 导师管理与联系状态维护
- 看板与统计视图
- 面试安排与复盘记录
- 标签、模板与提醒机制
- Markdown 风格在线笔记 / 文书编辑
- 资料专区与在线查看 / 编辑
- 学校 / 研究所检索与官方源库快照恢复
- 论坛交流、实时推送与 @ 提醒相关能力
- Docker 自托管与热发布脚本

## Operational Notes

- 默认数据库为 SQLite，适合个人或低并发场景
- 部署前请复制 `.env.example` 并替换为你自己的真实配置
- 如需快速上线，优先阅读 [DEPLOY.md](../../DEPLOY.md)
- 如需频繁迭代，优先使用 `pnpm deploy:hot:*` 链路
- 公开可体验地址为 <https://2320194668.cn>
