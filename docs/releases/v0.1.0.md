# v0.1.0 · First Public Open Source Release

发布日期：2026-03-08

## Release Summary

`v0.1.0` 是项目的首个公开开源版本，目标不是简单把代码推到 GitHub，而是把仓库整理成一个真正可阅读、可部署、可二次开发的公开仓库。

这次发布完成了三类工作：

1. 产品能力收口
2. 自托管部署链路固化
3. 公开仓库隐私与文档治理

## Highlights

### 1. 全流程复试 / 套磁工作台

项目已经覆盖从导师管理、面试安排、笔记沉淀到模板复用的一整套核心流程，适合保研 / 考研 / 复试准备场景持续使用。

### 2. 文书与资料管理能力

新增资料专区与在线编辑能力，使简历、成绩单说明、联系记录、复盘笔记等材料可以在统一工作台中维护，而不是散落在本地文件夹中。

### 3. 学校 / 研究所官方源库

支持学校与研究所检索，并提供官方源快照导入链路，方便在低内存服务器或离线场景中快速恢复院校基础数据。

### 4. Docker + HTTPS + 热发布

项目已固化 Docker Compose 部署、Caddy HTTPS 代理与免重建热发布方案，适合单机小服务器持续迭代。

### 5. 公开仓库准备完成

本次发布中已移除服务器信息、个人域名、私人联系方式、运维记忆等敏感内容，并补充正式 README、CHANGELOG 与 MIT License。

## Included Capabilities

- 邮箱登录 / 注册 / 重置密码
- GitHub OAuth 登录
- 导师管理与详情页维护
- 联系流程看板与统计视图
- 模板、标签、提醒、面试记录
- Markdown 风格在线笔记 / 文书编辑
- 资料专区与官方源库
- 论坛交流与实时推送相关能力
- Docker 自托管与热发布脚本

## Operational Notes

- 默认数据库为 SQLite，适合个人或低并发场景
- 生产部署前请替换 `.env.example` 中的示例值
- 如需快速上线，优先参考 `DEPLOY.md`
- 如需频繁迭代，优先使用 `pnpm deploy:hot:*` 链路

## Verification

本版本发布前已完成：

```bash
pnpm lint
pnpm build
```

并完成公开版敏感词复扫，确保仓库中不包含已知的服务器凭据、私有域名和私有联系方式。
