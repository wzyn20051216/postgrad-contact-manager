## [0.1.0] - 2026-03-08

### Added

- 支持邮箱注册、邮箱验证码找回密码与 GitHub OAuth 登录 / 注册流程
- 支持导师管理、联系状态跟踪、标签系统、模板管理与统计面板
- 支持面试安排、笔记管理、Markdown 风格在线编辑与资料专区能力
- 支持学校 / 研究所检索与“官方源库”快照导入恢复
- 支持论坛交流、提醒能力与实时推送相关链路
- 提供基于 Docker Compose + Caddy 的单机自托管部署方案
- 提供免重建热发布脚本，覆盖前端、后端、脚本与部署配置同步

### Changed

- 优化“学校 / 研究所官方源库”页面的后台检索体验与信息层级
- 优化公开版文档，将部署示例替换为通用域名 / 邮箱 / 服务器占位值
- 优化域名验收脚本，在系统 DNS 异常时增加 DoH 复核能力，降低误报率

### Security

- 移除公开仓库中的服务器信息、私有域名、联系方式与运维敏感配置
- 补充 `.gitignore` 规则，避免 `.env`、Agent 文件、临时测试文件被误提交

- 重写 README，补充项目定位、功能矩阵、部署说明、路线图与贡献指南
- 补充 `MIT` License
- 补充首发发布说明文档
